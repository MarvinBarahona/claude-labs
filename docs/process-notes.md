Worth creating one and codifying this pattern once a second lab actually reuses it (`structured-output-console` also uses `ModelPicker` and is the likely next confirming case) — not promoted to `docs/technical/` yet since this is still a single-consumer observation, not a confirmed cross-lab convention.
